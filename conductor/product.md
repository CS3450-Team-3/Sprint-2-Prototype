# Initial Concept
CodePop is a soda mixing application where users can create, order, and rate custom drinks. The project consists of a React Native mobile application and a Django backend.

# Product Guide: CodePop

## Vision
CodePop is the ultimate destination for soda enthusiasts to experiment with flavors, discover unique combinations, and share their creations with a community of like-minded "mixologists." By providing a seamless interface for customizing drinks with various sodas, syrups, and add-ins, CodePop turns a simple beverage into a personalized experience.

## Target Users
*   **Casual Drinkers:** Looking for a fun way to try new soda flavors.
*   **Flavor Experimenters:** Users who enjoy creating complex and unique drink recipes.
*   **Social Mixologists:** Users who want to share their favorite combinations and see what others are drinking.
*   **Store Managers/Staff:** Administrators who manage inventory, orders, and user feedback.

## Key Features
*   **Custom Drink Builder:** An intuitive interface to select a base soda and add multiple syrups and toppings.
*   **Social Catalog:** Browse, rate, and save drinks created by other users.
*   **AI Recommendations:** Personalized drink suggestions based on user preferences and past orders.
*   **Order Management:** A streamlined checkout process with cart functionality and payment integration.
*   **Admin/Manager Dashboards:** Tools for managing inventory (sodas, syrups, add-ins), processing complaints, and monitoring sales.
*   **User Profiles:** Manage personal preferences, saved drinks, and order history.

## Success Metrics
*   User engagement (number of drinks created and shared).
*   App store ratings and user feedback.
*   Efficient inventory management and order fulfillment.
